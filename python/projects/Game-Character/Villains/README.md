File: villains.py

Overview

This file defines all Villain character classes in the game. Each class inherits from Villain and represents enemies or playable villain characters.

Features
    •    Multiple villain classes:
    •    Necromancer
    •    DarkKnight
    •    Sorcerer
    •    Rogue
    •    Henchman
    •    Brute
    •    Overlord
    •    Unique enemy abilities
    •    Stronger scaling for later fights

Concepts Used
    •    Inheritance and specialization
    •    Enemy AI design
    •    Difficulty scaling

How to Use

Imported into the game:
from villains import Necromancer, DarkKnight

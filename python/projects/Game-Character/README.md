📂 Game_Character Project

Overview

This project is a terminal-based RPG battle game built in Python. Players choose to become a Hero or Villain, select a class, and fight through three battles while making choices that affect gameplay.

The project is structured using multiple files to demonstrate modular programming and object-oriented design.

Features
    •    Multiple character classes (Hero & Villain)
    •    Turn-based combat system
    •    Critical hits, block, and dodge mechanics
    •    XP and leveling system
    •    Story choices that affect gameplay
    •    Terminal-based UI with health/mana bars

Concepts Used
    •    Object-Oriented Programming (OOP)
    •    Inheritance and Polymorphism
    •    Encapsulation (properties & setters)
    •    Modular programming (multiple files)
    •    Randomization (game mechanics)
    •    Game loop logic

How to Run
cd Game_Character
python main.py

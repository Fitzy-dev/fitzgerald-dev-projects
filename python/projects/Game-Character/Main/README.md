File: main.py

Overview

This is the entry point of the program. It starts the game and connects all modules together.

Features
    •    Starts the game
    •    Handles player input (name, path)
    •    Runs battles
    •    Displays ending

Concepts Used
    •    Program entry point (if __name__ == "__main__")
    •    Modular execution
    •    High-level control flow

How to Run

Run this file to start the game:
python main.py

Salary Tracker Project

📌 Overview

This project is a Python class-based salary tracker that manages employee information such as name, level, and salary. It uses object-oriented programming principles like properties, setters, validation, and special methods to ensure employee data stays accurate and well-structured.

🚀 Features
    •    Creates employee objects with a name and level
    •    Assigns salaries based on employee level
    •    Validates names, levels, and salary updates
    •    Prevents invalid promotions or salary changes
    •    Uses getters and setters for controlled attribute access
    •    Includes __str__ and __repr__ methods for readable and reusable object output

🧠 Concepts Used
    •    Python classes and objects
    •    Encapsulation
    •    Properties and setters
    •    Validation with exceptions
    •    Class attributes
    •    Special methods (__str__, __repr__)


  ▶️ How to Run
    1.    Save the file as salary_tracker.py
    2.    Compile the program: g++ salary_tracker.py -o salary_tracker

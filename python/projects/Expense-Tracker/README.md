Expense Tracker (Python)

Overview

The Expense Tracker is a Python program that allows users to manage and analyze their spending across different categories such as food, transportation, and entertainment.

This project simulates real-world backend logic by organizing data using dictionaries and performing operations like adding, updating, and summarizing expenses.

⸻

Features
    •    Add expenses to categories
    •    Calculate total expenses per category
    •    Calculate total expenses across all categories
    •    Identify the highest spending category
    •    Handle invalid categories with error messages

⸻

Technologies Used
    •    Python 3
    •    Dictionaries
    •    Lists
    •    Functions

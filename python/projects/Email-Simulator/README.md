Email Simulator

Overview

This project builds a simple email simulator in Python using object-oriented programming. It models users, emails, and inboxes, allowing users to send, receive, read, list, and delete emails while tracking read status and timestamps.

Features
    •    Create User, Email, and Inbox classes
    •    Send emails between users
    •    Store received emails in an inbox
    •    List emails with numbered output
    •    Read full emails and mark them as read
    •    Delete emails by index
    •    Display timestamps for received emails
    •    Show email status as read or unread

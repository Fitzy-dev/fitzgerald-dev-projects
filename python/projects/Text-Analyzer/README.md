📝 Text Analyzer

📌 Overview

The Text Analyzer is a Python-based utility that processes and analyzes text data. It provides functions to count character frequency, detect anagrams, identify the most common characters, and extract unique characters. This project demonstrates foundational concepts in string manipulation, dictionary usage, and reusable function design.

⸻

⚙️ Features
    •    🔤 Character Frequency Analysis
Counts how many times each letter appears in a given text.
    •    🔁 Anagram Checker
Determines whether two strings are anagrams by comparing character frequencies.
    •    📊 Most Common Character
Identifies the character that appears most frequently in a string.
    •    ✨ Unique Characters Finder
Returns a list of characters that appear only once in the text.
    •    🧹 Punctuation Removal
Cleans text by removing punctuation while preserving letters, numbers, and spaces.


Planet Class

⸻

Overview

This project implements a Python class that models planets using object-oriented programming. Each planet has a name, type, and associated star, with built-in validation to ensure correct data. The class includes methods to display planet details and simulate orbit behavior.

⸻

Features
    •    Object-oriented design using a custom Planet class
    •    Input validation for data type and empty values
    •    Error handling with TypeError and ValueError
    •    orbit() method to describe planetary motion
    •    __str__() method for clean, formatted output of planet details
    •    Multiple planet instances to demonstrate functionality

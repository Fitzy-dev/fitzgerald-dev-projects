Overview

This project demonstrates how to build a basic Hash Table from scratch in Python without using Python’s built-in dictionary methods for hashing behavior. The program stores key-value pairs and uses a custom hashing function based on the ASCII values of characters.

This project was completed as part of the FreeCodeCamp Python Certification and focuses on understanding how hash tables work internally.

⸻

Features

Custom hash function using ASCII values
Add key-value pairs to the hash table
Remove key-value pairs
Lookup values by key
Collision handling using nested dictionaries
Dynamic storage structure

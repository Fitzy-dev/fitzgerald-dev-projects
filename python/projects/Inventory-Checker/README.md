Inventory Checker

⸻

📖 Overview

The Inventory Checker is a Python program that manages a collection of items and their quantities using a dictionary-based system. It provides functionality to add items, remove items, check stock levels, identify low-stock items, and calculate the total number of items in the inventory. This project demonstrates fundamental programming concepts such as functions, data structures, and input validation while simulating a real-world inventory management system.

⸻

🚀 Features
    •    Add new items or update existing item quantities
    •    Remove items with validation for stock availability
    •    Check if an item exists and view its quantity
    •    Identify items that are low in stock (less than 5 units)
    •    Calculate the total number of items in the inventory
    •    Case-insensitive item handling for consistency

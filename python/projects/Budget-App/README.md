Budget App

Overview

The Budget App is a Python project that models personal budgeting across multiple spending categories. It uses a Category class to manage deposits, withdrawals, transfers, and balance checks, and includes a spend chart function to visualize spending by category. This project demonstrates object-oriented programming, transaction tracking, and formatted output generation.

Features
    •    Create budget categories with separate ledgers
    •    Add deposits with optional descriptions
    •    Withdraw funds with balance validation
    •    Transfer money between categories
    •    Check available funds before spending
    •    Calculate current category balance
    •    Print a formatted ledger summary
    •    Generate a percentage-based spending chart by category

# Hangman Game (Python)

## 🎯 Description
A command-line Hangman game built in Python. The player guesses letters to uncover a hidden word before running out of attempts.

---

## 🚀 Features
- Random word selection
- Tracks correct and incorrect guesses
- Displays remaining attempts
- Allows full word guesses
- Simple text-based interface

---

## 🛠️ Technologies Used
- Python 3

---

## ▶️ How to Run

```bash
python3 hangman.py

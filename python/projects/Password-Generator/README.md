🔐 Password Generator (Python)

📄 File Name

password_generator.py

⸻

🧾 Overview

This program generates secure, customizable passwords based on user preferences. Users can choose the length of the password and select which types of characters to include, such as uppercase letters, lowercase letters, numbers, and special characters.

An additional feature allows users to include a themed word (such as places, foods, objects, or sports teams) to make the password more memorable while still maintaining security.

⸻

✨ Features
    •    Custom password length
    •    Optional inclusion of:
    •    Uppercase letters
    •    Lowercase letters
    •    Numbers
    •    Special characters
    •    Optional themed word integration (e.g., Pizza, Atlanta, Lakers)
    •    Prevents password from starting with a number or special character
    •    Password strength evaluation (Weak, Medium, Strong)
    •    Randomized password generation for better security

⸻

🧠 Concepts Used
    •    Randomization (random module)
    •    String manipulation (string module)
    •    Lists and list combination
    •    Conditional logic (if/elif)
    •    Loops (while, for)
    •    User input handling
    •    Basic cybersecurity principles (password strength)

⸻

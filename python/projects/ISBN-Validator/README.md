📚 ISBN Validator

📌 Overview

This project is a Python-based ISBN (International Standard Book Number) validator that verifies whether a given ISBN-10 or ISBN-13 code is valid.

The program:
    •    Accepts user input in the format: ISBN,length
    •    Validates input format and data types
    •    Calculates the correct check digit
    •    Compares it against the provided check digit
    •    Outputs whether the ISBN is valid or invalid

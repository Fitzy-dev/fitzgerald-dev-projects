📐 Polygon Area Calculator (Python)

📄 File Name

polygon_calculator.py

⸻

🧾 Overview

This program demonstrates object-oriented programming in Python by modeling geometric shapes using classes. It includes a Rectangle class and a Square class (which inherits from Rectangle).

The program allows users to calculate properties such as area, perimeter, diagonal length, and also visualize shapes using text-based graphics.

⸻

✨ Features
    •    Create and modify rectangles and squares
    •    Calculate:
    •    Area
    •    Perimeter
    •    Diagonal length
    •    Generate a visual representation of shapes using *
    •    Determine how many smaller shapes can fit inside a larger shape
    •    Demonstrates class inheritance (Square inherits from Rectangle)
    •    Custom string representations using __str__

⸻

🧠 Concepts Used
    •    Object-Oriented Programming (OOP)
    •    Classes and Objects
    •    Inheritance
    •    Method Overriding
    •    Encapsulation (getters and setters)
    •    Mathematical operations
    •    String manipulation
    •    Special methods (__str__)
    •    Main function structure (if __name__ == "__main__")

⸻

⚙️ How to Run the Program

1. python polygon_area_calculator.py

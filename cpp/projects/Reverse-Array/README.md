Reverse Array Project

📌 Overview

This project demonstrates how to reverse the elements of an array in C++ using pointers. It focuses on manipulating memory directly rather than using standard indexing, helping build a deeper understanding of pointer arithmetic and in-place data modification.

⸻

🚀 Features
    •    Reverses an array in-place using pointers
    •    Demonstrates pointer arithmetic (*, +, -)
    •    Avoids array indexing inside the function
    •    Efficient O(n) time complexity
    •    Includes before-and-after output for verification

⸻

🧠 Concepts Used
    •    Pointers in C++
    •    Dereferencing (*)
    •    Pointer arithmetic
    •    Functions and parameters
    •    In-place array manipulation


  ▶️ How to Run
    1.    Save the file as reverse_array.cpp
    2.    Run as g++ reverse_array.cpp -o reverse_array

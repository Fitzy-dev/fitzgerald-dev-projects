Square Function Project

📌 Overview

This project demonstrates a simple C++ function that calculates the square of a number using pass-by-value. It is designed to reinforce fundamental programming concepts such as functions, parameters, and basic arithmetic operations.

⸻

🚀 Features
    •    Computes the square of an integer
    •    Uses pass-by-value to keep the original variable unchanged
    •    Simple and easy-to-understand implementation
    •    Includes a sample function call in main()

⸻

🧠 Concepts Used
    •    Functions in C++
    •    Pass-by-value
    •    Basic arithmetic operations
    •    Input and output using iostream

▶️ How to Run
    1.    Save the file as square.cpp
    2.    Run as g++ square.cpp -0 square

Bank Account Project

📌 Overview

This project implements a simple banking system in C++ using object-oriented programming. It defines a BankAccount class that manages account details such as the account holder and balance, and provides functionality for deposits, withdrawals, and balance tracking. The project demonstrates encapsulation and class design principles.

⸻

🚀 Features
    •    Create a bank account with an initial balance
    •    Deposit funds into the account
    •    Withdraw funds with validation for sufficient balance
    •    Retrieve the current account balance
    •    Display account holder information and balance
    •    Uses encapsulation to protect account data

⸻

🧠 Concepts Used
    •    Classes and objects in C++
    •    Encapsulation (private and public)
    •    Constructors
    •    Member functions
    •    Input/output using iostream
    •    Basic validation logic
   
    ▶️ How to Run
    1.    Save the file as bank_account.cpp
    2.    Compile the program: g++ bank_account.cpp -o bank_account

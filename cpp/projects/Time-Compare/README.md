Time Compare Project

📌 Overview

This project implements a C++ program that compares two time values using a custom Time struct. It determines whether one time occurs earlier than another by comparing hours, minutes, and seconds in sequence. The project demonstrates structured data handling and logical comparison techniques.

⸻

🚀 Features
    •    Defines a Time struct with hour, minute, and second fields
    •    Compares two time values accurately
    •    Uses pass-by-reference for efficiency
    •    Returns a boolean result indicating which time is earlier
    •    Includes a sample test in main()

⸻

🧠 Concepts Used
    •    Structs in C++
    •    Pass-by-reference (const &)
    •    Conditional logic
    •    Function design
    •    Data comparison

    ▶️ How To Run
    1.    Save the file as time_compare.cpp
    2.    Compile the program: g++ time_compare.cpp -o time_compare
